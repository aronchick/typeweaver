Scenarios

These are the stress environments TypeWeaver should support first. The tool currently ships with a smaller starter set; this page makes the target set explicit.

Everyday web

Baseline reading on brighter screens with more forgiving contrast. Useful for first-pass comparisons.

Dark mobile

Smaller text, tighter layouts, darker interfaces, weaker contrast. Fonts start showing cracks fast here.

OCR / scan pressure

For capture, parsing, and machine-read workflows where character distinction matters more than style.

Dense dashboard

Tables, labels, controls, and secondary text all competing for attention in operational UI.
